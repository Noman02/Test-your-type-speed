Block Scope: A variable when declared inside the if else or switch conditions or inside for or while loops, are accessible within that particular condition or loop. To be consise the variables declared inside the curly braces are called as within block scope.
Global scope: The default scope for all code running in script mode.In addition, variables declared with let or const can belong to an additional scope
